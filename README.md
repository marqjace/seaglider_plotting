# split_sg_profile

Split Seaglider individual dive profile based on depth and create basic plots

- Temperature Profile,
- Salinity Profile,
- Conductivity Profile,
- T-S Diagram,
- Density Profile,
- Sigma-t Profile,
- Pressure Profile,
- Lat-lon Map,
- Buoyancy Profile,
- Heading Profile,
- Vertical Speed Profile,
- Glide Angle Profile,
